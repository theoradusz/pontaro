Inline Code

Use inline code for short snippets like variable names (myVariable), file paths (/etc/config), or commands (npm install).

Code Blocks

Here’s a JavaScript function:

function greet(name) {
  const message = `Hello, ${name}!`;
  console.log(message);
  return message;
}

// Call the function
greet('World');

Python example:

def fibonacci(n):
    """Generate Fibonacci sequence up to n terms."""
    a, b = 0, 1
    result = []
    for _ in range(n):
        result.append(a)
        a, b = b, a + b
    return result

print(fibonacci(10))
